BuzzTroll Mud on TiVo

BuzzTroll Mud

BuzzTroll Mud is a Java Mud client. It requires Java 1.4.2+.

  1. To run BuzzTroll Mud make sure you have Java 1.4.2+ properly installed (Java executable is in the path).
  2. Download and unpack the BuzzTroll Mud [tar.gz | zip]
  3. Update the sample configuration file (<btmud>/etc/sample.xml) and copy it to the $HOME/.btmudrc/ directory on Unix/Linux or %USERPROFILE%\.btmudrc\ directory on Windows. Make sure to at least update the Mud server information such as the Mud hostname, the port number, the username, and the password (the <server> element).
  4. Start the BuzzTroll Mud application by running: 
     <btmud>/bin/run

BuzzTroll Mud on TiVo

This is an experimental code.

BuzzTroll Mud on TiVo provides read-only Mud access. It's written with TiVo Home Media Engine SDK APIs.

  1. To run BuzzTroll Mud on TiVo first make sure that your TiVo DVR meets the requirements described on the HME Quick Start page.
  2. Download and install TiVo HME SDK 1.2.
  3. Ensure the samples included in the TiVo HME SDK work on your TiVo DVR as expected.
  4. Download and unpack the BuzzTroll Mud [tar.gz | zip]
  5. Update the sample configuration file (<btmud>/etc/sample.xml). At least update the Mud server information such as the Mud hostname, the port number, the username, and the password (the <server> element).
  6. Set the HME_HOME environment variable to the installation directory of the TiVo HME SDK.
  7. Start the BuzzTroll Mud application by running: 
     <btmud>/bin/runTivo <btmud>/etc/sample.xml
  8. Select the Mud application listed under the Music, Photos & More menu in your TiVo DVR.
